# IEEE BioInformatics Web Application (IEEE-EMBS Zewail City)

This project is a full-stack web application developed for the **IEEE-EMBS Club in Zewail City**.  
It serves as a platform for managing members, mentors, seminars, workshops, and announcements, while providing an intuitive and user-friendly interface for both administrators and participants.
